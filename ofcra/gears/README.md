# MODULE GEARS

Ce module permet de "_stuffer_" automatiquement les slots jouables par les humains: les infanteries seront mises en slip au lancement de la mission, puis en fonction de leur camp et de leur classe,
elles vont être équippées.

## Erreur
 Si la classe d'une infanterie n'est pas détectée, un message d'alerte est affiché dans le chat system.
